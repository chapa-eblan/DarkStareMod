DarkStare — Инструкция по сборке и настройке (Forge 1.20.1)
О моде
Название: DarkStare
Версия Minecraft: 1.20.1
Загрузчик модов: Forge 47.3.0
Тип: чисто клиентский хоррор-мод (не нужен на сервере).
Концепция: если игрок долго смотрит в абсолютно тёмный проход, экран начинает едва заметно «зумиться», звуки затухают, появляется гул/шепот и иногда мерцают глаза. Все эффекты плавные, атмосферные и настраиваемые.
Текущая структура файлов
В проекте /home/chapa/DarkStareMod уже есть готовый код:

src/main/java/com/darkstare/mod/ - DarkStare.java — главный класс мода; регистрация событий и конфига. - config/DarkStareConfig.java — конфигурация (Forge Config); все параметры эффектов. - client/ - DarkStareClientEvents.java — обработка тиков игрока, проверка темноты перед лицом, управление интенсивностью эффектов. - renderer/EyesOverlayRenderer.java — отрисовка мерцающих глаз в темноте (RenderLevelLastEvent). - renderer/SoundManager.java — управление звуками: гул ambient.end, шёпот ambient.cave, редкие «всплески».

src/main/resources/ - META-INF/mods.toml — мета-информация мода для Forge (id, версии).

build.gradle - текущий файл настроен под локальную сборку со stubs; для полноценной работы с Forge рекомендуется использовать официальный MDK.

Как правильно собрать мод с официальным Forge MDK (рекомендуемый способ)
Шаг A: Скачать MDK для 1.20.1

Открой сайт: Downloads for Minecraft Forge for Minecraft 1.20.1
Найди строку с версией Forge: 47.3.0 (или любую совместимую с 1.20.1).
Скачай файл: "mdk" (например: forge-1.20.1-47.3.0-mdk.zip).
Шаг B: Распаковать MDK

Создай новую папку, например: /home/chapa/DarkStareMDK
Распакуй скачанный mdk.zip в эту папку.
Проверь, что есть внутри:
build.gradle
gradle.properties
src/main/java/net/minecraftforge/example/...
src/main/resources/META-INF/mods.toml
Шаг C: Подготовить проект под DarkStare

В файле gradle.properties найди строки: mod_id=examplemod mod_name=Example Mod mod_version=0.1.0 И замени на: mod_id=darkstare mod_name=DarkStare mod_version=1.0.0

Удали папку примеров: src/main/java/net/minecraftforge/example/... src/main/resources/assets/examplemod/...

Скопируй из текущего DarkStareMod в новый проект:

ВСЮ папку: /home/chapa/DarkStareMod/src/main/java/com/darkstare В: (MDK)/src/main/java/com/darkstare Итоговая структура должна быть: src/main/java/com/darkstare/mod/ DarkStare.java config/DarkStareConfig.java client/ DarkStareClientEvents.java renderer/EyesOverlayRenderer.java renderer/SoundManager.java
Замени файл meta:

Открой (MDK)/src/main/resources/META-INF/mods.toml

Вставь содержимое из текущего: /home/chapa/DarkStareMod/src/main/resources/META-INF/mods.toml Или просто перепиши вручную:

modLoader="javafml" loaderVersion="[47,)" license="MIT"

[[mods]] modId="darkstare" version="${file.jar.version}" displayName="DarkStare" updateJSONUrl="" displayURL="Page not found · GitHub · GitHub" logoFile="" credits="" authors="chapa"

  [[dependencies.darkstare]]
      modId="forge"
      mandatory=true
      versionRange="[47,)"
      ordering="NONE"
      side="CLIENT"

  [[dependencies.darkstare]]
      modId="minecraft"
      mandatory=true
      versionRange="[1.20.1]"
      ordering="NONE"
      side="CLIENT"
Проверь build.gradle:

В нём уже есть блок minecraft { ... } из MDK — это и есть правильный конфиг.
Убедись, что runs.client.proserty/args не удалены случайно.
Шаг D: Первая сборка

В терминале перейди в папку нового проекта (MDK):

cd /home/chapa/DarkStareMDK chmod +x gradlew # если нужно ./gradlew clean build

Если всё ок, увидишь: BUILD SUCCESSFUL.
Скомпилированный jar будет по пути: build/libs/darkstare-1.0.0.jar
Шаг E: Запуск с модом (проверка)

./gradlew runClient

Игра запустится в режиме разработчика с твоем модом DarkStare.
Проверь эффекты:
Зайди в пещеру или сделай абсолютно тёмный коридор (свет 0).
Смотри туда несколько секунд — должен начаться зум, затухание звуков и появление гула/шепота, иногда мерцают глаза.
Настройка эффектов через конфиг
DarkStare полностью настраиваемый через Forge config:

.minecraft/config/darkstare.cfg

В нём можно:

Включить/выключить весь мод (general.enabled).
Настроить чувствительность к темноте (darkness_detection.*): rayLength, minConsecutiveDarkBlocks, maxLightLevel.
Управлять интенсивностью и скоростью нарастания эффектов (intensity.*).
Включить/выключить FOV-зум и микро-тряску (fov_zoom.*).
Настроить глаза: шанс появления, яркость, максимальное количество пар (eyes_overlay.*).
Настроить звуки: гул, шёпот, редкие всплески (sounds.*).
Включить/выключить виньетку по краям экрана (vignette.*).
Все параметры подробно описаны в DarkStareConfig.java.

Совместимость с другими модами
Мод:
Чисто клиентский (@OnlyIn(Dist.CLIENT) / DistExecutor), не ломает сервер.
Использует только стандартные события Forge (FOVUpdateEvent, RenderLevelLastEvent, PlayerTickEvent).
Не меняет ванильные классы и не патчит код — значит конфликты крайне редки.
Если ты ставишь его рядом с другими модами:
Он будет работать параллельно; его эффекты добавляются поверх существующего рендера.
